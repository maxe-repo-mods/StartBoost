# StartBoost

Customize run start: set starting level, currency, extra carts, and player upgrades (health, stamina, speed, etc).

R.E.P.O. BepInEx mod. Host-side only.

## Install

Requires [BepInEx 5.x](https://thunderstore.io/c/repo/p/BepInEx/BepInExPack/).

Place `StartBoost.dll` in `BepInEx/plugins/`.

## Configuration

Edit `BepInEx/config/maxenterme.StartBoost.cfg` after first launch.

## Build

```
dotnet build -c Release
```
